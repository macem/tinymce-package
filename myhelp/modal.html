TEXT PATTERN
are used to change the formatting of words and sentences

The default patterns is similar to markdown syntax so you can type:

*text* - makes text emphasis (1)

**text** - makes text strong (1)

??text?? - makes text in citation (1)

--text-- - makes text deleted (1)

# text - biggest heading 1 (2)

## text - bigger heading 2 (2)

### text - big heading 3 (2)

#### text - normal heading 4 (2)

##### text - small heading 5 (2)

###### text - smallest heading 6 (2)

1. text - makes text in ordered list (2)

* text - makes text in unordered list (2)

{quote} text - quote a block of text (2)

{code} text - makes a preformatted block of code (2)

NOTE:
1). inline patterns should work after we type *text* and space,
2). block patterns should work after we type #text and enter.
